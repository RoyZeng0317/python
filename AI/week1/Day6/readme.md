## 購物車介面

## 說明
我需要一個製作購物車的頁面，結合 HTML + CSS + JavaScript + python，
運用簡單的程式碼能夠做出不錯的介面產品

## 需求
01. 產品至少要有五項
02. 會運用到後端 /api 開發
03. 使用 pgAdmin 做資料庫連線
04. 運用到 pool 水池
05. 使用 python 的 flask 架構後端
06. .env建立好  我會親自填寫資料密碼，你不可以觸碰
07. 記得把 .gitignore 的內容做好安全檢查，避免暴露敏感資料在網路上
08. 僅可顯示區域內部網路，其他敏感資料不可外洩，端口我想放在 8097上
09. URL: http://localhost:8097/

## 頁面敘述

**首頁** 首頁內容要求可以看到店商平台的樣貌  可以到登入/註冊介面以及瀏覽五樣熱門商品
**商品頁** 該頁面可以列出在平台上的商品在頁面上
**購物車頁** 該頁面讓使用者可以在逛商品頁面時把喜歡的商品加入進去
**後台登入頁面** 管理員可以這個店商平台的後台管理系統頁面，可以
報表統計目前平台有多少使用者  每個使用者花費了多少錢
**結帳頁面** 使用者可以從購物車當中進行結帳
